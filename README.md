# ecommerce-sales-dashboard
An E-commerce Sales Dashboard created in Power BI to analyze yearly revenue, profit trends, and regional performance.
Project Overview
This project focused on transforming raw e-commerce transaction data into actionable business intelligence. The objective was to analyze sales trends and operational efficiency through robust data cleaning, SQL query management, and the development of an interactive dashboard.
What Was My Task?
Data Management: I was responsible for performing data cleaning to ensure data integrity.  Database Processing: My task involved writing and troubleshooting complex SQL queries to extract and aggregate key sales metrics.  Dashboard Development: I designed and built a comprehensive dashboard using Power BI to visualize sales performance.
What Was My Goal?Identify Trends:
The goal was to uncover patterns in sales performance and customer purchasing behavior.  Actionable Insights: I aimed to create a visual tool that allows stakeholders to make data-driven decisions regarding sales strategy.  Operational Clarity: The ultimate goal was to streamline reporting and provide a clear overview of e-commerce health.
